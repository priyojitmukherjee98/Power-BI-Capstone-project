# Retail Sales Analytics Dashboard

## 📌 Overview
This project focuses on building an interactive Power BI dashboard to analyze retail sales performance across products and categories.

## 🎯 Objective
To create a clear and interactive system for tracking:
- Revenue trends
- Product performance
- Return patterns
- Category-wise insights

## 🛠️ Tools Used
- Power BI
- Excel (Data Cleaning)

## 🔍 Key Steps
- Cleaned and structured raw sales data
- Built KPI metrics for performance tracking
- Designed interactive dashboards with filters and visuals

## 📊 Key Insights
- Identified top-performing products and categories
- Analyzed return trends affecting revenue
- Highlighted key revenue-driving segments

## 📈 Outcome
Enabled better decision-making by providing a centralized and visual performance monitoring system.
